claude-code

v2.1.126

Gateway model picker, project purge command, broader skip-permissions.

New `claude project purge` and expanded `--dangerously-skip-permissions` change your daily workflow hygiene and autonomy controls.

details

What changed:

  • `/model` picker now lists models from your gateway's `/v1/models` when `ANTHROPIC_BASE_URL` targets an Anthropic-compatible gateway
  • New `claude project purge [path]` wipes all Claude Code state (transcripts, tasks, file history, config entry) with `--dry-run`, `-y/--yes`, `-i/--interactive`, `--all`
  • `--dangerously-skip-permissions` now bypasses prompts for `.claude/`, `.git/`, `.vscode/`, shell config files, and other previously-protected paths (catastrophic `rm` still prompts)
  • `claude auth login` accepts pasted OAuth codes for WSL2 / SSH / container scenarios where browser callback can't reach localhost
  • OTel `claude_code.skill_activated` now fires for user slash commands with new `invocation_trigger` attribute (`user-slash`, `claude-proactive`, `nested-skill`)
  • Auto mode spinner turns red when a permission check stalls (no more mistaking a stuck prompt for active work)
  • Windows: PowerShell 7 detected from Microsoft Store, MSI-without-PATH, and .NET global tool installs; PowerShell treated as primary shell when its tool is enabled
  • Read tool: removed per-file malware-assessment reminder that triggered spurious refusals
  • Image paste >2000px now auto-downscales instead of breaking the session; oversized images in history are auto-removed and request retried

Breaking changes:

  • `--dangerously-skip-permissions` blast radius is now larger — previously-protected paths (`.claude/`, `.git/`, shell rc files) are no longer gated. If you scripted around the old behavior, audit it.
  • Security fix: `allowManagedDomainsOnly` / `allowManagedReadPathsOnly` were being ignored when a higher-priority managed-settings source lacked a `sandbox` block — review managed-settings layering if you rely on these.

Notable fixes:

  • API retry countdown actually counts down (no more stuck "0s")
  • "Stream idle timeout" after Mac sleep mid-request
  • False "Stream idle timeout" aborts in background/remote sessions during long thinking pauses
  • Hang where assistant finished thinking but produced no output after empty turns
  • Trackpad scroll speed in Cursor and VS Code 1.92–1.104 terminals
  • claude.ai MCP connectors suppressed by manual servers stuck in needs-auth
  • CJK garbled characters on Windows in no-flicker mode
  • Rare race clearing a valid OAuth refresh token

Links:

  • Release notes

Why it matters for you: Red stalled-permission spinner + project purge + paste-image auto-downscale are quality-of-life wins you'll feel daily; the wider `--dangerously-skip-permissions` scope is worth knowing before you next reach for it.
